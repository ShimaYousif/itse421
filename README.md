# itse421
هذه الصفحة لتحميل وثيقة متطلبات المستخدم الخاصة بك. قبل ان تكون قادرا على تحميل الوثيقة يجب عليك ان تقوم بعمل نسخة محلية على جهازك باستخدام الامر 
git clone https://github.com/nwesri/itse421.git
بعدها يمكنك نسخ ملف الوثيقة الخاصة بك في نفس الحافظة التي قمت بكتابة الامر العلوي بها وكتابة الامر التالي لاضافتها الى الريبوزتوري

git add docname.docx

سوف يتم اضافة الوثيقة الى الريبوزتوري ولكن لا يتم تحميلها بعد يمكنك في كل مرة ان تعلم الحالة الخاصة بالريبوزتوري الخاص بك عن طريق الامر

git status

بعد اضافة الوثيقة يجب اعتماد اضافتها الى الريبوزتوري وذلك باستخدام الامر

git commit

لرفع الوثيقة الى الريبوزوتوري على الانترنت استخدم الامر التالي

git push -u origin master
